Mutant Pizzas

Project Type: University Project
Frameworks Used: Unity
Languages Used: C#
Roles: Programmer

About

Mutant Pizzas is a Tower Defense game. After an experimental test done by scientist unicorns, some pizzas became mutants and decided to look for revenge against humanity for eating them. Now unicorns have to deffend the portal to the human world and prevent Mutant Pizzas from passing through it. Help the unicorns defend the portal that connects them with the human world from the mutant pizzas!

Roles

Programmer

As the only Programmer, I was responsible for making all the scripts for the game. In only three weeks I programmed the gameplay, architecture for multiple gamemodes, tools for making levels, UI backend, etc.
