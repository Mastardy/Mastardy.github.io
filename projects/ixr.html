IsraportXR

Project Type: Professional Project
Software Used: Unity
Languages Used: C#

About

IsraportXR is a real-time multi-user AR visualization tool for home decoration.

Roles

Developer

I designed and implemented all the core features, including interactive furniture and lighting placement, 3D notation tools, and real-time synchronization of user actions in multiplayer sessions.

To improve productivity, I created a cloud-based asset upload system using Rust, Amazon S3, and PostgreSQL, enabling seamless updates to furniture and material catalogs, significantly reducing client update times.
