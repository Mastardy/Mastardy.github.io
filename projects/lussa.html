The Final Frontier

Project Type: Professional Project
Software Used: Unreal Engine 5
Languages Used: C++
Roles: Developer

About

The Final Frontier is a multiplayer shooter that combines high-octane mobility with dynamic vertical combat. Set in a stunning sci-fi universe, players embark on an epic journey to secure humanity's survival amidst the chaos of a collapsing Earth. Featuring six unique character classes and evolving AI companions (PETS).

Roles

Developer

Fixed and optimized various graphics-related issues to improve visual fidelity and performance. Implemented NVIDIA Reflex to reduce system latency and DLSS for enhanced rendering efficiency. Integrated new weapons and character skills.
